# Matematikos užduotys

Šiame kataloge rasite matematikos užduotis suskirstytas pagal temas.

## Katalogai

| Katalogas | Paskirtis |
|-----------|-----------|
| **algebra/** | Tiesinės, kvadratinės, logaritminės lygtys ir sistemos |
| **geometrija/** | Geometrijos užduotys — plotai, perimetrai, įbrėžimai |
| **trigonometrija/** | Trigonometrinės lygtys ir tapatybės |
| **eksaminas/** | Egzaminų užduotys ir sprendimai |
| **kitos/** | Kitos matematikos temos |

## Failų formatas

- Kiekviena užduotis yra atskirame PDF faile
- Failo pavadinimas: `[skaičius]_[aprašymas].pdf`
- Pavyzdys: `100_uzduociu_sprendimas_x.pdf`

## Kaip pridėti naują užduotį

1. Sukurkite LaTeX šabloną pagal esamą formatą
2. Sukompiliuokite į PDF
3. Įkelkite į atitinkamą katalogą
4. Atnaujinkite `index.html` su nauja kortele
